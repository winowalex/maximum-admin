Vertical menu configuration

Verticle layout is default layout in some of theme types but if you want to change it then by changing data-layout="vertical" from other layout style the verticle layout can be set .

How to add new items in menu or make changes in menu items?

In order to add, update or remove menu items from the right side navigation, you can simply edit in src/layout/LayoutMenuData.tsx file. The change will be reflect on all the React files automatically.
