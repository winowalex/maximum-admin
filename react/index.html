Introduction

We are presenting you a fully responsive and React Redux Thunk with Toolkit, Bootstrap 5.3.2 template. It is a great choice for you in creating Admin, Dashboard, eCommerce Apps, CRM, Admin Panels, Real Estate Admin or SASS-based interface etc..

Steex is highly customizable and developer friendly Admin & Dashboard Template. Steex is made with firebase , fake Backend and Api integration.

Achieve growth and other ultimate benefit for your team to work faster and save your time. If you're looking for an admin dashboard which is fully responsive with Bootstrap and React, Redux then you can start your project using Steex Admin & Dashboard Template.

We have used Scss variables-based and gulp based modes. You can even change mode and layout by changing data attributes effortlessly instead of coverting source lines of codes in short time.

Steex has 5 dashboards, 9+ apps that can be used in Analytics, CRM, E-commerce, Learning, Real-Estate, File Manager, support or any other custom projects.

Steex has various themes ( Default, Minimal, Material, Creative, Modern, Interaction ) with single HTML code. You can change the theme by data attributes only no need to change whole code for another theme.


Start making Your Project with Steex

Firstly install your project , for installing please visit Installation page for more details how to install and run Steex.

We are suggesting to start with minimal setup and for that we have provided the Starterkit .

Steex has 150+ pages, 40+ components, 30+ charts, 30+ widgets. So, there is steps to be taken is to check the dependencies and clean the unrequired dependencies which is not necessary for your project.

After that, you can use the pages you wanted from whole project, and build your web.

Implementing Steex into your existing project

You can also use pages or codes in your existing project with installing related dependencies. Although it is possible to use Steex into existing product, it still require updation of code for better functionality.
