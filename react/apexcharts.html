Apexcharts

  1. Charts
  2. Apexcharts
Overview Official Website

React-ApexCharts is a wrapper component for ApexCharts ready to be integrated into your react.ts application to create stunning React Charts.

Add Package
yarn add apexcharts --save
Remove Package
yarn add apexcharts / remove specific package from package.json
